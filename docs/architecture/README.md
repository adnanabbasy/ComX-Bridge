# ComX-Bridge Architecture

This directory contains documents detailing the system design and architecture of ComX-Bridge.

## Available Documents

- **[Architecture Design](ARCHITECTURE.md)**: A comprehensive guide covering the system's core design, layer structure, interface definitions, and data flow.

## Overview

ComX-Bridge is designed as a modular, extensible communication hub.

- **Values**: Unified, Extensible, Intelligent, Open.
- **Layers**: Transport -> Packet Parser -> Protocol -> AI Engine -> Application.
- **Security**: Enterprise-grade security with JWT, RBAC, and mTLS.

Please refer to `ARCHITECTURE.md` for the full technical specification.
